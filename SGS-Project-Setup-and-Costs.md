# مشروع SGS — وثيقة التكلفة والأدوات والصلاحيات

> **مُوجّهة إلى:** إدارة شركة SGS / قسم تقنية المعلومات (IT)
> **مُعدّة من:** فريق التطوير
> **التاريخ:** 7 يوليو 2026
> **الغرض:** تحديد التكلفة الشهرية للمشروع، والبرامج المطلوب تثبيتها على اللابتوب المخصّص، والصلاحيات اللازمة لبدء العمل.

---

## 1) نظرة عامة

هذه الوثيقة توضّح متطلبات تشغيل مشروع SGS من ناحية **الاشتراكات الشهرية**، و**الأدوات والبرامج** التي يجب تثبيتها على اللابتوب الذي ستوفّره الشركة، و**الصلاحيات** المطلوبة من قسم تقنية المعلومات حتى يستطيع المطوّر العمل دون معوّقات.

**التقنيات الأساسية للمشروع:** React 19 + Vite + TypeScript على بيئة Node.js، مع قاعدة بيانات PostgreSQL عبر منصّة Supabase.

> **ملاحظة مهمة:** تكلفة **استضافة السيرفر (Server Hosting)** مؤجّلة للشهر القادم (تُدرَج لاحقاً بعد تحديد نوع السيرفر — انظر القسم 6). الأرقام أدناه تغطّي أدوات التطوير والاشتراكات فقط.

## 5) الصلاحيات والوصول المطلوب من قسم تقنية المعلومات (IT)

حتى يعمل المطوّر بكفاءة، نطلب توفير الصلاحيات التالية على اللابتوب المخصّص:

1. **صلاحيات مسؤول محلي (Local Admin)** لتثبيت وتحديث البرامج أعلاه.
2. **وصول كامل للتيرمنال** (PowerShell / Windows Terminal) **دون قيود أو حظر** — بما يسمح بتشغيل أدوات سطر الأوامر وتغيير سياسة التنفيذ (Execution Policy).
3. **تشغيل Claude Code داخل التيرمنال** بدون منع، مع إمكانية **تشغيل حتى 3 وكلاء فرعيين (Sub-agents)**.
4. **تثبيت وربط أدوات MCP** (خوادم MCP) بالمشروع بحُريّة.
5. **صلاحية رفع الكود (Push) إلى GitHub** — إنشاء/الوصول لمستودعات الشركة والرفع عليها.
6. **صلاحية استخدام Docker** — بناء الصور (Images) وتشغيل الحاويات (Containers) دون حظر.
7. **صلاحيات Google Chrome** الكاملة (إضافات التطوير، أدوات المطوّر).
8. **فتح الوصول الشبكي (Outbound Network)** للنطاقات التالية على الأقل:
   - `registry.npmjs.org` (حزم npm)
   - `github.com` (رفع الكود)
   - `hub.docker.com` (حاويات Docker)
   - `api.anthropic.com` (خدمة Claude Code)
   - `supabase.com` (قاعدة البيانات)
   - `higgsfield.ai` (توليد الصور/الفيديو)

> باختصار: **لا نريد أي أداة مقفلة أو محظورة** من الأدوات المذكورة، لأن العمل بالكامل يتم عبر التيرمنال وهذه الأدوات.

## 6) نوع السيرفر (مطلوب توضيح — التكلفة مؤجّلة للشهر القادم)

نرجو من قسم تقنية المعلومات **تزويدنا بنوع السيرفر المُستخدَم داخل الشركة**، مثل:

- سيرفر داخلي (On-Premise) — نظام Linux أو Windows Server؟
- أو سيرفر سحابي (Cloud) — AWS / Azure / GCP / VPS؟
- المواصفات المتاحة (المعالج، الذاكرة، مساحة التخزين، نظام التشغيل).

> **تكلفة استضافة السيرفر تُدرَج في وثيقة الشهر القادم** بعد تأكيد النوع والمواصفات.
